FACULTYGUARD WEBSITE — REVISED PACKAGE

Upload the contents of this folder to the root of the FacultyGuard Netlify site or GitHub repository.

Primary file: index.html
Styles: styles.css
Images: assets/
SEO pages: six separate HTML files
SEO support: sitemap.xml and robots.txt
Netlify configuration: netlify.toml

The package uses only relative local paths, so CSS and images will load when the entire folder is uploaded together.
