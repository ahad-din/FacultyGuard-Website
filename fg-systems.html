FG Systems

Practical infrastructure for the full teaching cycle.

Each FG System addresses a defined area of faculty practice with implementation support and resources included. Select one focused system or connect several around a broader need.

Seven Systems

Designed around the semester as faculty experience it.

FG System 1

Semester Startup

Build the operating foundation for a new term before predictable demands begin.

  • Policy and course routine alignment
  • Communication expectations
  • Documentation setup
  • Resources included for repeatable launch

FG System 2

Student Communication

Create calm, consistent pathways for routine, sensitive, and high-volume communication.

  • Message categories and response standards
  • Escalation and follow-up pathways
  • Boundary-setting language
  • Resources included for recurring situations

FG System 3

Faculty Documentation

Maintain useful records of decisions and interventions without over-documenting.

  • Decision and exception records
  • Student concern timelines
  • Consistent note structure
  • Resources included for faculty records

FG System 4

Academic Integrity

Use an evidence-led workflow that supports fair review and defensible judgment.

  • Concern review sequence
  • Evidence organization
  • Communication and documentation flow
  • Resources included for consistent review

FG System 5

AI Policy

Clarify acceptable use, authorship, disclosure, and assignment-level expectations.

  • Course-specific AI position
  • Assignment use categories
  • Disclosure expectations
  • Resources included for student guidance

FG System 6

Grading & Feedback

Create a sustainable evaluation rhythm while preserving faculty discretion.

  • Rubric and criteria alignment
  • Feedback sequence
  • Consistency review
  • Resources included for recurring feedback

FG System 7

Semester Closeout

Finish records, capture lessons, and prepare the next term without rebuilding from zero.

  • Record completion
  • Recurring issue review
  • System retention decisions
  • Resources included for next-term preparation

System Selection

FacultyGuard helps determine whether one system is enough or whether several should be coordinated.
